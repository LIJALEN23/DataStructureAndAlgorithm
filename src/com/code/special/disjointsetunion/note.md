# 并查集(Disjoint Set Union : DSU)

## 基本概念与术语
### 概念
- 概念：**并查集** 是一种树型的数据结构，用于处理一些不相交集的 **合并(union)** 及 **查询(find)** 问题
- 思想：用一个 *数组* 表示了整片森林(parent)，树的根节点(root)标识了一个集合，通过找到某个元素的根节点就可以确定它在哪个集合中。  
### 术语
- **查找(find)**：查找元素所在的集合
- **合并(union)**：将两个集合合并起来
- **判断连接(isConnected)**：判断两个元素是否在同一个集合
